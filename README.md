# 🔐 Password Generator

Gerador de senhas seguras desenvolvido em Python.

## 📌 Sobre o projeto

Este projeto cria senhas aleatórias utilizando letras maiúsculas, letras minúsculas, números e caracteres especiais.

Foi desenvolvido para praticar programação em Python e conceitos relacionados à criação de senhas mais seguras.

## 🚀 Funcionalidades

- Geração de senhas aleatórias
- Escolha do tamanho da senha
- Uso de letras, números e símbolos
- Criação de senhas mais fortes

## 🛠️ Tecnologias utilizadas

- Python
- VS Code
- Git
- GitHub

## 📚 Conceitos praticados

- Variáveis
- Strings
- Estruturas de repetição
- Funções
- Bibliotecas `random` e `string`
